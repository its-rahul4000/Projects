Problem Statement:



**Forecasting of Off Shore Wind Power:**



**Objectives:**

· Review off shore wind power forecasting methodologies from international and Indian perspectives.

· Evaluate advantages, limitations, and applicability of different forecasting approaches.

· Develop an effective wind power forecasting model using an appropriate programming platform.

. Validate forecasting accuracy using real wind generation data from an RE-rich site



**Major Deliverables:**

1\. Literature review on wind power forecasting.

2\. Comparative assessment of forecasting techniques.

3\. Forecasting model implementation.

4\. Validation results using actual generation data.

5\. Final project report.
