## Temperature Sensor Casing 

Round cyllindrical metallic casing was used to case temperature sensor. 

### Dimensions of Box - 

Diameter: 4.2 mm
Height: 1.6 mm

### Challenges Faced - 

**Finding a Compact Metallic Sensor Casing**
* One of the key challenges was sourcing a small and compact metallic casing suitable for housing the small temperature sensor.
* Metallic 3D printing was not available, so a ready-made component had to be used instead.
* The selected casing had to closely match the design requirements to ensure proper integration with the sensor and overall enclosure.
